# About

repo that tracks my journey to learn bazel (7.4.1) via interactive sessions with
claude and grok (when i run out of claude tokens)

# Structure

i'll start with minimal knowledge and have question and answer sessions with the AI's to teach me the principles/patterns in bazel.

during every session, i'll branch from master (session/001), put an entry in `docs/qa/SESSION_N.md` that records my 'QA' session with the ai, and any notes (`docs/notes`) that might guide future sessions.

i will push all branches (or maybe make tags) so anyone who cares (mostly my future self) can step through them to see how the chain of understanding is built up.
